# SellSense Prototype

A prototype for a real-time sales assistant that uses emotion and tone detection to improve virtual communication.

## 💡 MVP Goals
- Real-time webcam emotion detection
- Microphone tone/sentiment analysis
- Live suggestions to the seller based on emotional/tone input
- Basic UI using Streamlit

## 📁 Project Structure
- `app.py` - Main Streamlit interface
- `emotion_detector.py` - Handles webcam-based emotion detection
- `tone_analyzer.py` - Handles microphone tone analysis
- `models/` - Pre-trained or custom emotion/tone models
- `utils/` - Utility functions
- `data/` - Sample recordings or logs

## ▶️ Getting Started

```bash
pip install -r requirements.txt
streamlit run app.py
```
